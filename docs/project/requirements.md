# FinTrack

## Overview

FinTrack is a personal finance management application that helps users manage their income, expenses, monthly budgets, savings goals, and financial insights. The application is built using a client-server architecture with a Next.js frontend and a Laravel REST API backend backed by PostgreSQL.

---

## Objectives

- Secure user authentication
- Track income and expenses
- Manage custom categories
- Set monthly budgets
- Track savings goals
- Visualize financial data using charts
- Generate monthly financial summaries
